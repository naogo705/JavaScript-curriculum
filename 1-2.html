JavaScriptについて

 

JavaScriptとは

JavaScript は、クロスプラットフォーム（異なるOSやデバイスなどのプラットフォームにおいて同一アプリケーションを動かす仕組みのこと）で動作するオブジェクト指向のスクリプト言語で、ウェブページをインタラクティブにするために使用されます (例えば、複雑なアニメーション、押下可能なボタン、ポップアップメニューなどを設けること) 。
Node.js のような、より高度なサーバー側のバージョンの JavaScript もあり、ファイルをダウンロードするだけでなく (複数のコンピューター間でのリアルタイムの連携など) 、ウェブサイトに多くの機能を追加することができます。ホスト環境 (例えばウェブブラウザ) 内では、JavaScript とその環境のオブジェクトが結びつけられ、プログラム制御が可能になります。

 

JavaScriptとJavaの違い

そもそも「JavaScript」と「Java」は全く違う言語となっており、
「プログラミング言語である」と「同じ単語が入っている」ということくらいしか共通点がないほど、用途から書き方まで全てが違います。

それぞれの違いについて4点ほど挙げときましたので目を通しておきましょう。

1.開発した会社

まず、それぞれ開発された会社が違います。
「JavaScript」はNetscape Communications社が開発し、「Java」はSun Microsystems社が開発しております。

 

2.使用目的

「JavaScript」は容易に開発できる手軽さと素早さを求めて開発されており、
「Java」は「Write Once, Run Anywhare（一度プログラムを書けば、どんなハードウェアでも動作する）」ことを目指して開発されました。

 

3.文法

文法の違いとしては、「JavaScript」ではデータ型というものを適宜変更できる形式（動的型付）を取ります。
一方、「Java」ではデータ型を最初に固定で決める形式（静的型付）を取ります。
また、「Java」はコンパイル（コンピューターが処理できる形に翻訳すること）が必要であるが、JavaScriptはコンパイルが必要ありません。

 

4.利用される分野

「JavaScript」はWebブラウザ上で色を変えたり、ポップアップ表示させるといった動きがあるWebページを開発する際に主に使われております。
そして、「Java」は業務システムやTwitterなどのWebサービス、スマートフォン用（Android）向けアプリなどを開発することができます。

 

JavaScriptとJavaの名前が似ている理由

「JavaScript」と「Java」は全く別物なのに、なぜ名前が似ているかというと、
そもそも「JavaScript」は、開発当初はLiveScriptという名前でした。
しかし、「Java」が発表された当時、世界中のユーザーから大人気だったため、その人気に便乗しようと名前を「JavaScript」に変更したのが理由です。

そのため、名前が似ているので同じようなプログラミング言語だと誤解しないようにしましょう。
