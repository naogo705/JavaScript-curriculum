関数について

 

関数（function）とは

関数（function）とは、様々な処理を1つにまとめて、名前をつけることができるものです。単調な処理を1つにまとめて、どこからでも使えるように効率化するという目的でよく使われます。
1つにまとめることで、同じ処理を何度も書く必要が無くなり、ミスが減る上、別のプログラムに使いまわすことも可能になります。
また、関数（function）は値を受け取ったり、何らかの処理を行った値を返すことも可能なので、効率のよいプログラミングには必須のスキルと言っても過言ではないです。

 

関数（function）の使い方

functionを利用するには、一般的に関数宣言、関数式、コンストラクタという3種類の方法があります。

 

1.関数宣言によって記述する方法

一番ポピュラーな方法なので、見たことがある人も多いかもしれません。
これは文字通り、関数をそのまま宣言することでプログラム内で利用することができる方法です。

　　function sample() {
　　　ここに処理を記述する
　　}

この例では、「sample」という関数名を付けて関数宣言を行なっています。
関数名は自分で勝手に決めて大丈夫ですが、一般的には「どんな処理をするのか？」というのが名称から推測できることが望ましいです。
例えば、何らかの値を取得するのであれば「get○○○()」、値を追加するのであれば「add○○○()」というように決めていくとよいでしょう。

 

2.関数式で記述する方法

2つ目は、関数式（関数リテラル）によるfunctionの利用方法です。
関数を宣言する際に関数名を記述しなくても良いので、無名関数や匿名関数とも呼ばれる方法になります。

　　let sample = function() {
　　　ここに処理を記述する
　　}

この例では、関数名を指定せずにそのまま変数へ式を代入する形式で記述しているのが分かります。実はJavaScriptの関数は、文字列や数値と同じくデータの値として存在しています。 そのため、このような書き方が許されるというわけになっております。

 

3.コンストラクタで記述する方法

3つ目は、コンストラクタを利用したfunctionの記述方法です。
この方法では、JavaScriptに標準で用意されているfunctionオブジェクトのコンストラクタを利用して関数を定義していきます。

　　let sample = new Function('text', 'console.log(text)');

この例では、第1引数に引数名として「text」を指定して第2引数に処理の内容を記述しております。
あとは、呼び出す際に「sample('hello')」と記述すれば引数のtextに「hello」という文字列が格納されてコンソールに出力されるという仕組みです。

 

基本的な関数（function）の作り方

前述では、functionの使い方として「関数宣言」や「関数式」などを見てきました。
それでは、実際に関数を作成するときはどのように行うのかを見ていきたいと思います。

　　let p = document.createElement('p');
　　p.textContent = "Hello World!";
　　document.body.appendChild(p);

これは、単純なp要素をJavaScript側で作成して画面に表示するプログラムです。

createElementメソッドでHTMLのpタグを生成しており、それを"p"という変数に代入しております。
textContentプロパティはノードおよびその子孫のテキスト情報を取得・設定するために使うものです。そのため、この例の場合だと、生成したpタグのテキスト情報を"Hello World!"に設定しているということになります。
そして、最後にdocument.body.appendChild(p);というのはbodyタグの子要素として、生成したpタグを設定するという記述になっております。

これを関数化したい場合には、適当な関数名を付与して{ }で囲んであげれば良いわけです。

　　function createP() {
　　　let p = document.createElement("p");
　　　p.textContent = "Hello World!";
　　　document.body.appendChild(p);
　　}

この例では、「createP」という関数名を付与しております。これにより、以降は呼び出す際に「createP()」と記述するだけでp要素を作成して表示することができるようになります。
もちろん、HTMLタグの属性「onclick」に記述すればクリックすることで関数を実行させることもできます。

 

関数（function）の呼び出し方

関数は宣言したあとに呼び出すことで、はじめて利用することができます。ここでは「呼び出し」という表現を使っていますが、簡単に言えば定義した関数を実行するという意味になります。

基本的な呼び出し方法や、よく使われるonclick()やonload()による呼び出し方も紹介したいと思います。

1.基本的な呼び出し方法

　　function sample() {
　　　let text = "Hello World!";
　　　console.log(text);
　　}
 
　　sample();

この例で作成している「sample関数」は、文字列をコンソールログに出力する機能を持った関数になります。
そして、この関数を呼び出すには関数名をそのまま利用して「sample()」と記述すればOKです。しっかりと文字列が出力されているかはご自身で実践して確認してみてください。
ちなみに、この呼び出し方法は関数式を使ったfunctionの場合でも同じなので忘れないようにしてください。

 

2.クリックしたら呼び出す方法：onclick()

今度はHTML要素のonclick属性を使って関数を呼び出してみます。
onclick属性を使用することで、指定した要素がクリックされた際にfunctionを動作させることができます。

例えば、先ほど作成した関数「sample()」をonclick属性に当てはめると次のようになります。

　＜button onclick="sample()"＞ボタン＜/button＞

この例では、button要素のonclick属性に関数「sample()」を指定しているのが分かります。これにより、ボタンをクリックした場合のみ関数のsample()が実行されるということになります。
つまり、onclick属性に指定した関数は、JavaScriptで定義されている関数を自動的に探し出して実行してくれるというわけです。

 

3.ロードしたら呼び出す方法：onload()

次に、onload()メソッドを使った関数の呼び出し方法です。
これを使うと、HTMLファイルなどのリソースがすべて読み込まれたタイミングで、任意の関数を実行することができます。

　　window.onload = function() {
　　　conslo.log("Hello World!");
　　}

この例では、「window.onload」に関数をそのまま代入しているのが分かります。
この記述によりHTMLファイルなどが読み込まれたあとに関数が実行されるということになります。

 

関数(function)の引数

関数(function)で処理を1つにまとめられるようになってきたら、引数を指定して処理を効率化していきましょう。
ここからは、一般的な引数の利用方法を詳しく見ていきたいと思います。

1.引数を設定する方法

一般的な記述方法としてはfunction( 引数, 引数・・・ )のようにいくつでも引数を記述することができます。
この引数は、関数内で自由に扱えるようになっています。処理の中に組み合わせて使うことが可能です。

　　function sample(name, age) {
　　　この中に記述する処理で引数（nameとage）が使用できる
　　}

関数sample()の引数として「name」と「age」を記述しているのが分かります。
あとは、この関数内で「name」と「age」を普通の変数みたいに扱うことができるようになります。

 

2.関数の引数に値を渡す方法

次に、設定した引数に何らかの値を渡す方法について紹介していきます。
例えば、先ほど引数として「name」と「age」を設定しましたが、これらに値を渡すには次のようにします。

　　function sample(name, age) {
　　　console.log(name + 'さんの年齢は' + age + 'です！');
　　}

　　sample("一郎", 25);

実行結果：「一郎さんの年齢は25です！」

最後のsample()に引数を指定している箇所に注目してください！
「sample("一郎", 25)」と引数を記述することで、それぞれ順番に「name / age」へ値が格納されていきます。つまり、「name」には「一郎」という文字列が入り、「age」には「25」という数値が格納されているということになります。
これらの引数を活用すれば、実行結果のように1つの文字列として出力することも出来るようになります。

引数を使用すれば、以下のようにさまざまな値にも応用することが可能です。

　　sample("一郎", 25)
　　sample("次郎", 23)
　　sample("三郎", 20)

実行結果：「一郎さんの年齢は25です！」「次郎さんの年齢は23です！」「三郎さんの年齢は20です！」

1つの関数を作るだけで、さまざまなケースに対応できるようになるのが引数の大きな特徴であるということです。

 

関数（function）の戻り値（返り値）

関数は戻り値（返り値）を設定することで、関数内で処理した結果を返すことが可能になります。
そこで、一般的に使われるreturn文を利用したプログラミング手法について見ていきたいと思います。

 

1.return分の記述方法

関数内にreturn文を記述することで、任意の値を返すことができます。
「値を返す」と言ってもイメージしにくいかもしれませんが、これは簡単に言うと、何らかの処理をした結果を「プログラムに伝える」ということです。

　　function multiply(num1, num2) {
　　　return num1 * num2;
　　}

　　console.log( multiply(5, 2));

実行結果：10

この例では、「multiply()」に2つの引数が設定されており、実行時に数値を当てはめています。処理の内容としては、引数「num1」「num2」を掛け算してからreturnしているだけです。
ここでreturnを記述するだけで、掛け算をした結果を返すようになります。
その証拠に、console.log()の中で関数をそのまま実行していますが、実行結果には掛け算の結果が出力されております。
つまり、関数が実行されたあと、処理内容の結果をconsole.log()へ伝えているというわけです。

 

2.戻り値を使って条件分岐を行う方法

return文を活用するとif文などの条件式にも応用することが可能です。

　　if (multiply(5, 2) > 5) {
　　　console.log("5よりも大きいです。");
　　} else {
　　　console.log("5よりも小さいです。");
　　}

実行結果：5よりも大きいです。

先ほど作成した「multiply()」は2つの引数を掛け算した結果を返す関数でした。
そこで、if文の条件式にmultiply()をそのまま当てめることで、「10 > 5」という条件式と同じになるわけです。
このようにreturn文を使えば、関数を普通の値として活用できるようになるので応用範囲が広くなるというわけです。

 

関数（function）のスコープ

関数(function)で扱う変数は「スコープ」に注意する必要があるので、合わせて紹介しておきます。
そもそも「スコープ」というのは変数が利用できる範囲のことで、関数内で宣言した変数と関数外で宣言した変数では扱える範囲が異なります。

　　let num1 = 10;
　　function sample() {
　　　let num2 = 100;

　　　console.log(num1);
　　}

　　sample();

　　console.log(num2);

実行結果：「10」
　　　　　「num2 is not defined」

この例では、変数「num1」が関数外で宣言されており、変数「num2」が関数内で宣言されている点に注目してください。
関数「sample()」を実行するとコンソールに「num1」の値が出力されます。「num1」は関数外で宣言された変数ですが、これにより関数内でも利用できることが分かると思います。
逆に関数内で宣言された「num2」は関数外で利用しようとしてもエラーになってしまっていることが分かると思います。
つまり、関数内で宣言された変数は関数内でしか利用できず、関数外で宣言された変数は関数内でも利用できるというわけです。

これらを「ローカル変数（関数内で宣言）」「グローバル変数（関数外で宣言）」と呼びます。
