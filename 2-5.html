JSONデータ形式、配列について

 

JSONデータ形式

JSON（ジェイソン）とは「JavaScript Object Notation」の略で、その名前の通りオブジェクトを表記するための書き方の決まりごとです。もともとはJavaScriptと連携する他の言語とのやりとりをするために考え出されたものです。しかしJavaScriptだけでなく、複数の言語間のやりとりでも使えるため、さまざまなシーンでとても重宝されています。
{}で囲い、「キー」と「値（バリュー）」というのを並べるような型で繋いでいくデータ形式のことを言います。

　　let apple = {
　　　"name": "りんご",
　　　"price": 200
　　}

上記の例で言うと、nameという「キー」に対して、"りんご"という「値（バリュー）」が設定され、priceという「キー」には"200円"という「値（バリュー）」が設定されています。

JSONの便利な点は、カンマ（,）で区切って複数のデータを定義することが可能なところです。

JSONのわかりやすい参考記事になります。確認してみてください。

 

配列について

配列とは、関連性のある「値」を1つの変数に代入したい時に使うものです。

配列はJavaScriptの中でもよく使われており、 配列とは値に順序をつけて格納できるオブジェクトのことです。 配列に格納したそれぞれの値のことを要素、それぞれの要素の位置のことをインデックス（添え字）と呼びます。 インデックスは先頭の要素から0、1、2のように0からはじまる連番となります。

配列の作成には（[と]）の中に要素をカンマ（,）区切りで記述するだけです。

　　let fruits = ["りんご", "ぶどう", "いちご", "みかん", "もも"]

　　let numbers = [1, 2, 3, 4, 5]

作成した配列の要素へインデックスとなる数値を、配列[インデックス]と記述することで、 そのインデックスの要素を配列から取得することができます。 配列の先頭要素のインデックスは「0」となります。

そのため、console.log(fruits[0])と記述するとコンソール上には「りんご」と表示されます。
numbersという配列の"3"をコンソールに表示したい場合は、console.log(numbers[2])と記述すれば表示することができます。このように配列名、インデックス番号を指定するだけで対象の要素を簡単に取得することができるということです。

JSON形式で配列を定義する場合は、

　　let fruits = [
　　　{
　　　　"name": "りんご",
　　　　"price": 200
　　　},
　　　{
　　　　"name": "ぶどう",
　　　　"price": 250
　　　}
　　]

JSONデータ間をカンマ（,）で区切ることによって複数の要素をデータとして配列の中に組み込むことが可能です。

 

チャレンジ課題

ではJSON、配列、if文を使って、課題にチャレンジしてみてください。

  • console.log()を使用し、コンソールに下記を表示させてください。
    りんごの金額は200円です。
    ぶどうの金額は250円です。
    いちごの金額は300円です。
    りんごとぶどうといちごの合計金額は750円です。
  • if文を使用し、コンソールに「いちごはぶどうとりんごより高額です。」と表示させてください。
