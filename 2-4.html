if文について

 

if文

if文とは「何か条件に従った時にだけ、あるプログラムを実行する」というものです。

例文で紹介すると、

　　let number = 15;

　　if (number > 10) {
　　　　console.log(number + "は10より大きいです");
　　}

numberという変数に代入された15という数字が10より大きい場合はif文の中身のプログラムが実行されて「15は10より大きいです」とコンソール上に表示されます。
このnumberという変数に代入されている数字を10より小さい数字にしている場合は条件を満たしていないためif構文の中身のプログラムは実行されませんので、例文の場合だと何もコンソール上には表示されません。

if文の解説をしていきます。
まずif()として()内には条件式を記述します。
そして{}内に条件が真（true）だった場合の実行内容を記述します。

 

else、else if

elseとは、条件が成立していない（偽 = false）だった場合の実行内容を記述できるものです。条件が偽（false）であった場合の処理を命令するには、次のように「else」を用いて条件を追加します。

例文で紹介すると、

　　let number = 5;

　　if (number > 10) {
　　　console.log(number + "は10より大きいです");
　　} else {
　　　console.log(number + "は10より小さいです");
　　}

この場合、条件が一致する場合と一致しない場合で異なる処理が行なわれます。では、条件分岐をさらに増やしたい場合はどう記述すればよいのでしょうか。if文にはelseの他に「else if」というフロー制御があります。

　　let number = 7;

　　if (number > 10) {
　　　console.log(number + "は10より大きいです");
　　} else if (number > 5) {
　　　console.log(number + "は5より大きいです");
　　} else {
　　　console.log(number + "は5より小さいです");
　　}

このようにelseとelse ifを使用することで条件分岐を複数設定することが可能となります。

 

比較演算子について

比較演算子はif構文の条件分岐で頻繁に使用します。比較演算子には「型を揃えた比較」と「厳密な比較」の2種類があり、場合によって使い分けを行ないます。

型を揃えた比較

記法 意味
== 　　　　　左右が等しい場合にtrueを返す
!= 　　　　　左右が等しくない場合にtrueを返す
< 　　　　　左辺より右辺が大きい場合にはtrueを返す
> 　　　　　左辺より右辺が小さい場合にはtrueを返す
<= 　　　　　左辺より右辺が大きいか等しい場合にはtrueを返す
>= 　　　　　左辺より右辺が小さいか等しい場合にはtrueを返す

最もよく使用されるのは「==（等価演算子）」です。この等価演算子の特徴として、左右の型が違う場合、同じ型に変換してから評価を行ないます。
「==」の場合、文字列の"1"と数字の1を比較するとtrueが返されます。

厳密な比較

記法 意味
=== 　　　　　左辺と右辺が型、内容ともに等しい場合はtrueを返す
!== 　　　　　左辺と右辺が型、内容いずれかが等しくない場合はtrueを返す

左右を、型を含めて比較する「厳密な比較」もあります。型と内容の両方を評価して真偽を返すため、型が違う場合は常に偽（false）を返します。
「===」の場合、文字列の"1"と数字の1を比較するとfalseが返されます。なぜならデータ型が違うからです。

 

論理演算子について

JavaScriptで使用できる論理演算子には、AND、OR、NOTの3種があります。

記法 意味
&& 　　　　　AND 左右の式がともにtrueの場合、trueを返す
|| 　　　　　OR 左右の式どちらかがtrueを返す
! 　　　　　NOT 式がtrueの場合はfalseを返し、falseの場合はtrueを返す

論理演算子を使用した記述方法を紹介しときます。

　　　console.log("aは1、かつbは1ではない");
